Welcome

This Academic Course Organizer is designed to help students manage their university courses in a simple and organized way. It allows you to store important course information such as instructors, credit hours, and personal notes.

Purpose of the Website

The purpose of this website is to help students stay organized throughout the semester. Instead of keeping scattered notes, everything can be stored in one place.

Features

You can view your courses in a table format, add new courses easily, and keep notes for each subject.

Why It Matters

Organization is a key factor in academic success. This project shows how basic HTML and CSS can be used to build a useful real-world tool.
